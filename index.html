About Me

Hi, I'm Usman, the creator of Talk Flow — a platform designed to make AI conversations smoother, faster, and more intuitive. Talk Flow started with a simple idea that communication should feel natural and effortless. With a focus on clean design, responsive performance, and real user experience, my goal is to make every interaction effortless and engaging. I'm passionate about building projects that combine logic, creativity, and simplicity — and Talk Flow is a reflection of that vision. Thank you for visiting and being part of the flow.


Talk Flow is a dynamic chat application that connects to the Google Gemini API to deliver real-time conversational responses. This project showcases my ability to integrate external APIs, manage asynchronous operations efficiently, and build seamless, responsive user experiences. It reflects my skills in handling real-time data, structuring clean API calls, and creating intuitive, modern web applications.
